# 墨客实验室
墨客实验室是一个地球上的非营利组织。我们以协作的方式创造工具，让所有人都能**享受探索科学与技术的乐趣**。

| 项目 | 描述 | 状态 |
|---------|------|--------|
| [墨干编辑器](https://mogan.app) | 基于GNU TeXmacs的结构化编辑器 | 稳定 |
| 墨干语言 | 一门结构化的编程语言 | 设计中  |
| 墨客星球 | 墨客社区的公共文档 | Beta |
| 墨者百科 | 科技领域的百科 | Alpha |
